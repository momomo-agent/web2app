web2app

Turn any web content into native iOS & Android apps. One command.

$ git clone → npm install → npm link

Three modes

🌐

URL

Wrap any website into a native app. Just pass a URL.

📄

File

Single HTML file → full native app with offline support.

📦

Project

React, Vue, vanilla — any web project, auto-detected.

Usage

URL mode 
# Wrap a website into an Android app
web2app init --url https://your-site.com --name "My App" --platform android

# Build APK
cd my-app && web2app build --platform android
File mode 
# Turn an HTML file into an app (both platforms)
web2app init --source ./index.html --name "My App" --platform both
Project mode + permissions 
# Web project with camera & mic permissions
web2app init --source ./my-project/ --name "My App" \
  --permissions camera,microphone,location \
  --fullscreen --orientation portrait

Features

📱 iOS + Android

Build for both platforms from a single project. APK and IPA output.

🔒 Permissions

Camera, microphone, location, storage, notifications — declare what you need.

📡 Online & Offline

URL mode for live sites, file/project mode for fully offline apps.

🎨 Customizable

App icon, splash screen, status bar color, orientation lock.

⚡ Zero config

Auto-detects build output (dist/, build/, public/). No config files needed.

🤖 Agent-friendly

Designed for AI agents. Claude Code, Cursor, OpenClaw — just paste the docs.
